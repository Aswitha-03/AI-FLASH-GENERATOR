# AI Smart Flashcard Generator

An advanced AI-powered study tool that converts notes into interactive flashcards. Features include:

- User Authentication (Login & Signup)
- AI-based Flashcard Generation
- Interactive Quiz Mode
- Study Analytics Dashboard
- Bookmark/Favorites System
- Dark Mode Support
- Progress & Streak Tracking
- Flask Backend + Modern Frontend UI

Built using Flask, JavaScript, HTML, CSS, and NLP techniques to enhance learning and retention.
